# flappy_bird
Установить Node.js - https://nodejs.org/en
Ввести команду: npm create vite@latest flappy-react
Выбрать: React, JavaScript
Добавить файл Game.jsx и директорию game в директорию src
Директорию img в public
Перейти в папку cd flappy-react
Установить зависимости: npm install
И запустить: npm run dev
Перейти в браузере: http://localhost:5173/
